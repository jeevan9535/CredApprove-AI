# 💳 CredApprove AI

CredApprove AI is a simple machine learning project that predicts whether a customer will get credit approval based on their financial and personal details.

The goal of this project is to simulate how banks make basic credit decisions using data like income, family size, and other factors.

---

## 🚀 What this project does

- Takes customer details as input
- Predicts approval or rejection
- Shows risk score (how risky the customer is)
- Calculates an estimated credit limit
- Displays simple insights based on inputs

---

## 🛠 Technologies used

- Python
- Pandas
- Scikit-learn
- Streamlit
- Machine learning 

---

## 📊 Model Information

For this project, I used machine learning classification models to predict whether a customer will be approved for credit.

The main model used is:

- Random Forest Classifier

### Why this model?

Random Forest was chosen because:
- It works well with structured/tabular data
- Handles multiple features effectively
- Gives good accuracy without heavy tuning
- Reduces overfitting compared to a single decision tree

### Output

- Predicts: Approved or Rejected
- Also provides probability, which is used as a risk score
